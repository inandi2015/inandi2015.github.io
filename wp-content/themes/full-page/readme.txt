== Description ==

SKT Full Page is full wide page slider which is simple, flexible, easy to use, WooCommerce friendly. Industries like food, recipe, chef, restaurant, bistro, cafe, model, portfolio, resume, IT software computers, maintenance, events, concerts, hotel, resort, motel, health, fitness, yoga and gym. WordPress 5.0 gutenberg editor friendly.
